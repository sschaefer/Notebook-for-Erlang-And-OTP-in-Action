resource_discovery is an exercise deisgned to introduced distributed applications.

Intended Use

resource_discovery is designed to receive "I have", "I want", and "Trade knowledge" messages to maintain a list of available "resources" which can be queried by a "fetch resources" messages. See system_tests/0 function in resource_discovery_tests.erl for a demonstration of some subtleties.
